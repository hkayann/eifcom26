# EIFCOM 2026 Website

This repository contains the source code for the International Workshop on Mobile Systems with Efficient Foundation Models (EIFCOM 2026) website.

The site is built using Hugo and the Ananke theme.

## Prerequisites

- Hugo (extended version recommended)
- Git

## Local Development

1. Clone the repository:
   git clone --recursive https://github.com/hkayann/eifcom26.git

2. Run the Hugo server:
   hugo server

The site will be available at http://localhost:1313/

## Deployment

The site is automatically deployed to Netlify when changes are pushed to the main branch.
URL: https://eifcom26.netlify.app/
